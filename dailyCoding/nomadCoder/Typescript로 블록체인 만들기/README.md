# README

## Install
1. 빌드 환경 설정
```shell
npm init
# yarn init
```
2. TypeScript 설치
```shell
npm install typescript --save-dev
# 전역
# npm install -g typescript

# yarn
# yarn global add typescript
```
3. tsc-watch 설치
   
```shell
npm i -g tsc-watch
```
3. 컴파일
- `package.json` 참고
```shell
npm start
```
